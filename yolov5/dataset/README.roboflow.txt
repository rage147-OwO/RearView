
P2_Dhaka_Dataset - v29 2022-05-21 7:46pm
==============================

This dataset was exported via roboflow.ai on May 21, 2022 at 1:48 PM GMT

It includes 5080 images.
Roadobjects are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Resize to 640x640 (Stretch)

The following augmentation was applied to create 5 versions of each source image:
* 50% probability of horizontal flip
* Random brigthness adjustment of between -25 and +25 percent
* Random Gaussian blur of between 0 and 1 pixels
